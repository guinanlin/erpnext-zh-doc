Frappe行业资源索引

A curated list of awesome things related to the Frappe Framework. Maintained by Gavin D'souza.

Inspired by awesome-django.

免责声明: 列出的项目可能是第三方社区包。不得审查或认可的贡献者。在使用之前检查每个项目的兼容性信息。你可以自愿使用它们。

介绍​

Frappe 是一个完整的栈，包含电池，用 Python 和 Javascript 编写的 web 框架，以 MariaDB (还有 Postgres)作为数据库。这是一个支持 ERPNext 的框架，非常通用，可以用来构建数据库驱动的应用程序。

应用​

Apps that showcase the power of the Frappe Framework

商业应用​

  • Apparelo - 面向服装行业的制造工作流管理。
  • Cargo Management - 用于ERPNext的包裹管理应用。
  • ERPNext - 开源的全功能业务管理系统。
  • Expenses - 用于ERPNext的费用管理模块。
  • FiMax - 用于ERPNext的贷款管理和还款计划。
  • Frappe Desk - 设计精良的开源工单系统。
  • Frappe HR - Frappe生态系统的开源人力资源管理系统。
  • Frappe Insights - 适用于Frappe应用的免费开源数据分析工具。
  • Gameplan - 远程团队的协同开源工作沟通工具。
  • Healthcare - 专为医疗行业打造的开源管理系统。
  • POS Awesome - 使用Vue.js和Vuetify为ERPNext打造的开源销售点系统。
  • PropMS - 由Aakvatech提供支持的基于ERPNext的房地产管理解决方案。
  • Restaurant - 用于ERPNext的餐厅应用。
  • School - 驱动 mon.school 和 frappe.school 的学习管理系统 (LMS)。
  • ServiceMS - 由Aakvatech提供支持的用于ERPNext的服务管理系统。
  • Stone Warehouse - 管理按批次结算的余额，专为陶瓷瓷砖供应商而设。
  • TailPOS - 针对ERPNext的线下首选开源销售点系统。
  • One-FM - 有很多可以学习的自定义项目， 印度人开发的
  • 费用项目 - 针对财务的费用做专项项目

工具​

  • Alerts - 登录后向特定收件人显示自定义警报。
  • Background Tasks Unleashed - 一个Frappe任务调度和自动化。
  • Bulk Webhook - Bulk Webhook 允许创建发送多个记录和报告的 ERPNext Webhook，并支持 Apache Kafka。
  • Chat - 适用于 Frappe 部署的现代聊天工具。
  • ERPNext OCR - 在 Frappe 中使用 Tesseract 的光学字符识别。
  • ERPNext Quota - 用于管理 ERPNext 站点、用户、公司和空间限制的应用。
  • Frappe ReST API Wrapper - 在 Frappe 之上构建自定义 ReST API。
  • Frappe System Monitor - Web 界面用于监视 web 服务器运行的进程和系统利用情况（RAM、CPU、磁盘）。
  • Go1 CMS - 基于 Frappe 构建的先进内容管理系统。
  • Photos - 基于 AI 的图像分类和标记，类似于 Google Photos 中的 Desk。
  • Pibicut - 带有 QR Code 生成器的 URL 缩短工具。
  • PibiDAV - 与 NextCloud 服务器集成 WebDAV、CalDAV 和 CardDAV（未来），用作（DMS），用于在上传文件到 Frappe 时将 Frappe 文件上传并标记到 NextCloud 的复制。
  • Raven - 专为 Frappe 构建的简单开源团队消息平台。
  • Release - 管理 Frappe 和 Frappe 应用的发布。
  • Sentry - 将错误日志发送到 Sentry 以进行调试。
  • Sheets - 在线电子表格应用和 ERPNext 之间的轻松同步。
  • Temporal - 一个与 Redis 集成的 ERPNext 应用，快速提供日历信息。
  • ToolBox - 使用 ToolBox 自动化您的站点维护。
  • Wiki - 用于提供动态数据以及内置审阅系统的 Wiki。
  • ERPNext POS Hardware Integrations - 秤重秤集成、Stripe Terminal 集成和通过 QZ Tray 的原始打印。

插件​

  • Active User Lister - 显示当前活跃用户列表的应用。
  • Attachment Control extended - 提供更多对字段级附件的控制的插件。
  • Database Console - 直接在 Frappe/ERPNext 桌面执行 SQL 查询，就像使用 'bench mariadb' 一样。
  • Desk Navbar Extended - 对 Frappe 的导航栏稍作修改。
  • DFP External Storage - 适用于 Frappe 和 ERPNext 的每个文件夹的 S3 兼容外部存储管理应用。
  • ERPNext: Fiscal Year - 使日期相关字段遵循默认财政年度的开始和结束日期的桌面插件。
  • ERPNext: POS Restrictions - 帮助在默认 POS 中添加一些限制的 ERPNext 插件。
  • Frappe tinyMCE - 将 Frappe 的 Quill 文本编辑器替换为 tinyMCE 文本编辑器。
  • Jodit HTML Editor - 在 Frappe/ERPNext 版本 11 和 12 的网页和表单中将 Quill 文本编辑器替换为 Jodit HTML 编辑器。
  • Language Translator - Frappe 桌面和网站的自动语言翻译工具。
  • Language Toggle - 在 ERPNext 的导航栏上添加一个按钮以切换语言。
  • List View extended - 带有更多自定义选项的列表视图插件。
  • List View: Unassign From - 为桌面的列表视图添加取消分配多选支持的 Frappe 插件。
  • Numeric Control extended - 允许在网站和桌面上更多地控制数字字段。
  • OIDC Extended - ERPNext 社交登录认证方法（OIDC）的扩展，具有新功能以满足企业需求。
  • S3 Attachments - 用于在 Frappe 中存储和检索文件的 S3 存储桶插件。
  • Select Control extended - 为选择控件添加选项组支持的插件。
  • Silent Print - 用于静默打印文档的实用工具，即无需与浏览器的打印对话框交互，直接将打印指令发送到打印机。
  • Strict Session Defaults - 强制并管理会话默认弹出框。
