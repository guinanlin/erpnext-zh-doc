开源

ERPNext 是免费和开源的， 基于 GNU General Public License (GPLv3)](https://www.gnu.org/licenses/gpl-3.0.en.html) 授权. 基于此授权，您将获得：

  • 基于任何目的使用软件的自由
  • 自由更改软件以满足您的需求
  • 与任何人共享软件的自由
  • 分享您所做的更改的自由

ERPNext的源代码存放于 GitHub.

开源软件的优点:

  1. 无须绑定供应商.

  2. 可以在任何位置部署应用程序, 对服务器拥有自主权， 并独自拥有私有数据.

  3. 你可以通过 社区 寻求帮助.

  4. 因为产品被广泛使用， 您可以从使用被众多的批评和使用者的反馈中受益，他们报告了数百个问题和改进产品的建议。 虽然 ERPNext 是免费和开源的，但为 ERPNext 运行服务器、实施和支持它并不是一件容易的事。 您可以在 erpnext.com 上选择专业帮助或寻找合适的服务提供商。 这将使您能够降低风险并专注于提供业务价值。
