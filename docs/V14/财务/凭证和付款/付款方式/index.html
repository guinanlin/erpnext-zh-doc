付款方式

付款方式是进行或接收付款的媒介。

要访问付款方式列表，请访问：

主页 > 会计 > 设置 > 付款方式

1. 如何创建支付方式 ​

  1. 转到付款方式列表，然后单击新建。

  2. 输入付款方式的名称。

  3. 设置类型是现金、银行还是一般。这对于了解销售点 (PoS)中使用的支付方式很有用。

  4. 为所有公司设置默认付款帐户。

  5. 保存。

提示：设置默认帐户会将此帐户提取到付款条目中。

注意：在进行付款条目时，如果设置了默认银行帐户，将按以下顺序获取：

  * Company form
  * Mode of Payment default account
  * Customer/Supplier default bank account
  * Select manually in Payment Entry
