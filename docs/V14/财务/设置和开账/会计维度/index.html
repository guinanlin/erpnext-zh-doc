会计维度

维度会计意味着为每个交易打上适当的维度标签，如分支机构、业务单元等。这样可以让您单独维护每个部分，从而限制总体对GL账户的维护，并使您的科目余额表保持纯净。

成本中心和项目在ERPNext中默认被视为维度。在设置会计维度字段后，在适用的交易报告中将添加该字段。

在ERPNext中，您可以创建可配置的会计维度并在交易和报告中使用它们。要访问“会计维度”列表，请转到：

首页 > 财务 > 设置 > 会计维度

1. 创建会计维度 ​

  1. 转到会计维度列表，然后单击“新建”。

  2. 选择您要用作自定义维度的参考文档。例如，如果您选择部门作为参考文档，则该维度将基于部门。

  3. 输入维度名称（此名称将显示在创建维度的交易中）。在“默认尺寸”表格内，您可以像下面的屏幕截图所示那样提及公司特定的默认尺寸。该尺寸将自动获取并应用于相应公司的交易中。

  4. 如果要使该维度成为记账环节必填项，请勾选“强制执行”复选框。

2. 用法 ​

当您创建维度时，将使用特定维度的后台作业创建自定义字段。您可以在会计维度部分中看到它们，这部分对会计凭证产生影响。

您还可以根据这些维度过滤各种财务报告，如损益表、资产负债表和总账。

2.3 使会计维度对“损益表”和“资产负债表”账户强制执行 ​

收益和损失”是代表一段时间内您的会计交易的“收入和支出账户”的组合。

资产负债表账户包括“资金运用（资产）”和“资金来源（负债）”，它们表示公司在任何给定时间的净值。

通过选择复选框“必须适用于利润与损失账户”或“必须适用于资产负债表”，您可以将维度配置为对“利润与损失账户”和“资产负债表账户”强制执行。

2.4 禁用会计维度 ​

如果您不再需要它们，也可以禁用维度。原有的会计维度的旧纪录将保持不变。
