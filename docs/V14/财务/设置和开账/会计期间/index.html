会计期

会计期间定义了财务报表被记录的时间段。

在 ERPNext 中，业务交易(如销售/购买发票、股票分录、工资分录、日记分录等)只允许在规定的会计期间内创建。

为什么需要会计期？

当交易提交时，它们影响分类账和处理分类账数据的报告。如果必须编制财务报告供当局审计或结清本财政年度的会计账簿，这可能会造成问题。

在这里，会计期限可以用来限制交易提交的时间，以保持相应报告的完整性。

1. 如何创建会计期 ​

1.1 选定交易的「封闭」选项是什么? ​

交易文档类型的子表中的“封闭”选项用于选择在会计期结束后哪些文档类型将受到限制。

请注意，如果会计期结束，如果在子表中的任何选定的交易没有“已关闭”检查，那么他们将不会受到会计期结束后的限制。

  1. 输入会计期间的名称。

  2. 通过设置开始和结束日期来定义时间框架。

  3. 从表中添加或删除事务。请注意，所有交易列表中与“封闭”选项勾选将在会计期间结束后受到限制。

  4. 保存和提交。

如果您试图在会计期结束后提交一个已关闭的交易，您将看到一个验证错误，阻止您这样做。

注意: 任何角色都不能提交会计期间定义的交易，即使是“允许设置冻结账户和编辑冻结条目的角色”中设置Account Settings.的角色也不能提交。
