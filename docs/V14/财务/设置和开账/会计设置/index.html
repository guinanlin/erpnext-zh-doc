账户设置

在ERPNext中有各种帐户设置，可以限制和配置会计模块中的操作。

账户设置 #

1. 超额计费 (%) #

Over Billing Allowance

您可以超额计费的百分比。例如，如果订单价值为100美元，项目中设置的百分比为10％，则允许您计费110美元。

2. 允许超额角色 #

Role Allowed to Over Bill

具有此角色的用户可以在允许百分比之上进行超额计费。

3. 允许绕过信用额度的角色 #

Role allowed to bypass Credit Limit

选择允许提交超过设定信用额度的交易的角色。信用额度可以在客户表单中设置。

4. 检查供应商发票号码的唯一性 #

Check Supplier Invoice Number Uniqueness

勾选后，将不允许使用相同的‘供应商发票号’录入采购发票。这对于避免重复输入非常有用。

5. 取消发票时取消付款链接 #

Unlink Payment on Cancellation of Invoice

如果勾选此项，系统将取消与相应发票的付款链接。默认情况下，如果提交了付款条目，则必须先取消关联的发票才能取消该付款条目。在取消链接后，您现在可以取消和修改发票。但是这些支付不会被链接并且被视为预付款。

6. 自动从订单中获取付款条款 #

Automatically Fetch Payment Terms from Order

启用此功能将自动从采购/销售订单中获取付款条款，并应用到其关联的采购/销售发票。

7. 在删除交易时删除会计和库存分类账目录条目 #

Delete Accounting and Stock Ledger Entries on deletion of Transaction

启用此功能将允许在删除发票和收据时删除关联的总账和库存分户账目。如果您不想在取消文档后丢失文档ID，可以勾选此项。现在，您可以取消并删除文档以再次获得相同的文档ID。

8. 自动计算资产折旧入账 #

Book Asset Depreciation Entry Automatically

勾选后，将根据设置的第一个日期自动创建资产折旧条目。例如，基于资产主数据中预定的折旧次数，在接下来的3/4年内安排物品每年的折旧。有关更多详细信息，请访问“资产折旧 ”页面。

9. 取消订单时解除预付款链接 #

Unlink Advance Payment on Cancellation of Order

与之前的选项类似，这个选项会取消任何针对采购/销售订单所做的预付款。

10. Enable Common Party Accounting #

If checked, an adjustment Journal Entry will be posted automatically on creation of Sales/Purchase Invoices against common Customer & Supplier. For more details, visit Common Party Accounting

11. 为变更金额创建分类账目条目 #

Create Ledger Entries for Change Amount

如果勾选了，对于销售发票，系统将考虑给定的找零金额来发布分类帐分录。

12. 折扣记账 #

Enable Discount Accounting

如果勾选此项，销售发票的项目表中可以添加折扣账户，这将使您更有效地记录应用于项目的折扣。它还允许您为项目添加默认折扣账户，在将项目添加到销售发票时会自动获取该信息。

税设置 #

1. 从地址确定税收类别 #

税收类别 可以在地址上设置。一个地址可以是发货地址或账单地址。设置应用税收类别时要选择哪个地址。

2. 自动从商品税模板中添加税费和费用 #

如果为某个项目设置了 计税模板 ，并且在交易中选择了该项目，则启用此功能将填充交易中的税表。

关闭期间设置 #

1. 账户冻结至某日 #

将会计交易冻结到指定日期，除了指定的角色之外，任何人都不能进行/修改条目。

2. 允许设置冻结账户和编辑冻结日记账的角色 #

具有此角色的用户可以设置冻结账户并针对冻结账户创建/修改会计分录。

递延账户设置 #

1. 递延日记账凭证计算基础 #

递延收入金额可以根据两个标准进行预订。这里的默认选项是“天”。如果选择了“天”，则递延收入金额将基于每个月的天数进行纪录；如果选择了“月份”，则将基于每个月的数量进行纪录。例如：如果选择了“天”并且需要在12个月内递延$12000的收入，则30天的那个月将会被分配$986.30，31天的那个月将会被分配$1019.17。如果选择了“月份”，则无论一个月有多少天，都会在每个月中递延$1000 的推迟收入。

2. 自动处理递延日记账凭证 #

此设置默认启用。如果您不希望自动发布延期会计分录，可以禁用此设置。如果禁用了此设置，则必须手动处理延期会计，使用处理递延会计功能。

3. 登记递延日记账凭证 #

默认情况下，分户账条目会直接发布到预收款账簿以抵消发票。为了通过日记帐输入此递延金额的帖子，可以启用此选项。

4. 自动入账 #

此选项仅适用于通过日记账录入的待核算分录。默认情况下，待发布的日记账条目保持草稿状态，用户必须手动验证这些条目并提交它们。如果启用了此选项，则会自动提交日记帐条目而无需任何用户干预。只有在勾选确认递延分录时才会显示此选项。

打印设置 #

1. 在打印中显示含税价 #

应用的税费将显示在打印预览中。

2. 在打印中显示付款计划 #

使用 付款条款时，可以看到“付款计划表”。启用此功能将在打印视图中显示该表。

汇率设置 #

1. 允许使用历史的汇率 #

如果您希望ERPNext检查外汇交易中从货币兑换获取的记录的年龄，则应取消选中此项。如果未选中，文档中的汇率字段将为只读。 过期天数是在决定货币兑换记录是否过期时使用的天数。

当“允许历史汇率”被禁用时，这是有效的。因此，如果将Stale Days设置为10，则可以接受10天以内的历史汇率。如果启用了Allow Stale Rates，则对于历史汇率没有时间限制。

​ 如果启用了历史汇率，则获取顺序如下：

  1. 从货币兑换表单获取最新价格

  2. 如果找不到货币兑换表单，则自动获取市场上最新价格

    如果禁用了陈旧汇率，则获取顺序如下：

    1. 从货币兑换表单获得最新价格，在“Stale Days”设置范围内。
    2. 如果找不到任何货币兑换单据, 则自动获取市场上最新价格

报表设置 #

1. 使用自定义现金流表格式 #

You may choose to use Custom Cash Flow Formats to customize what the Cash Flow report looks like. To know more, visit this page.

您可以选择使用自定义现金流格式来定制现金流报告的外观。了解更多信息，请 访问此页面. 。
