Docker配置Nvm-Node-yarn

在 Ubuntu 上安装 Node.js 和管理不同版本的 Node.js 最方便的方式是使用 Node Version Manager（NVM）。

以下是在 Ubuntu 上安装 NVM 并安装 Node.js 18 的步骤：

  1. 安装 NVM： 打开终端并运行以下命令来下载并运行 NVM 安装脚本：

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

    或者，如果你更喜欢使用 wget：

    wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

    这会下载 NVM 安装脚本并执行它。按照终端中的提示操作即可。

  2. 重启终端或执行以下命令： NVM 安装完成后，为了使 NVM 命令生效，你需要重启终端，或者可以运行以下命令：

    source ~/.bashrc

    如果你使用的是 zsh 或其他终端，请根据自己的情况调整命令。

  3. 安装 Node.js 18： 安装 NVM 后，可以使用以下命令来安装 Node.js 18：

    nvm install 18

    这会下载并安装 Node.js 18 版本。

  4. 使用 Node.js 18： 安装完 Node.js 18 后，你可以使用以下命令来切换到这个版本：

    nvm use 18

    或者，你也可以设置默认版本为 Node.js 18：

    nvm alias default 18

  5. 验证版本： 运行以下命令来验证 Node.js 和 npm 的版本：

    npm --version

  6. 安装yarn： 运行以下命令安装yarn：

    npm install -g yarn

确保显示的版本号是 Node.js 18 和相应版本的 npm。

请注意，NVM 提供了方便地切换不同版本的 Node.js 的功能。

你可以使用 nvm ls 命令来查看已安装的 Node.js 版本，并使用 nvm use <version> 来切换版本。

如果在安装或使用过程中遇到问题，请查阅 NVM 的文档或查找相关社区资源来获取更多帮助。
