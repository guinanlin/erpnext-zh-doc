安装指南

ERPNext 针对不同的需求， 有不同的建议安装方式。

需求建议安装方式注意事项
开发，体验，培训docker compose 一键安装
生产环境： 单机版ubuntu 22.0 纯手工安装

docker compose 一键安装

方法1：

镜像地址： https://hub.docker.com/r/goodhawk/erpnext14

1， 此镜像为 ERPNext14 版本，定期更新到官方最新版本

2， 适合开发，体验， 培训 

version: '3.3'
services:
    erpnext14:
        ports:
            - 4070-4075:80-85
        privileged: true
        volumes:
            - 'ERPNextDB01:/var/lib/mysql'
            - 'ERPNextAPP01:/home/frappe/frappe-bench/sites'
        container_name: ent01
        image: 'goodhawk/erpnext14:basic'
volumes:
  ERPNextDB01:
  ERPNextAPP01:
